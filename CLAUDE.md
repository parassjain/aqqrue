# Project Context

I want to build and ai workflowagent.
Goal of the AI agent is take input of any csv, read the instructions from the chat and do the task on the csv.

Once the user gives the csv and instructions, the agent should be able to understand the task, plan the steps needed to accomplish it, and execute those steps on the csv file. The agent should also be able to handle any errors or issues that arise during the process and provide feedback to the user.

# Rules

- Always ask clarifying questions before starting a complex task
- Show your plan and steps before executing
- Keep updating the plan as you progress and learn more about the task
- Keep the original csv file unchanged and work on a copy of it
- keep separate input and output files in different folders to avoid confusion
- Create tools to alter the csv file as needed, such as filtering, sorting, and aggregating data.
